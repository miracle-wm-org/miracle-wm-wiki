# Enable Animations
Globally enable or disable animations.

## Example
```yaml
# ~/.config/miracle-wm/config.yaml

enable_animations: false  # disable animations globally
```

## Schema

```yaml
enable_animations: <boolean>
```

## Properties

### `enable_animations`

:   <small>required</small> **type:** Boolean

    When `true`, animations are enabled globally. When `false`, all animations are disabled.

## Default
```yaml
enable_animations: true
```
